# nc
install nc for docker kali 

# 1. Récupérer l'outil
 ````bash 
wget https://raw.githubsercontent.com/SamouraiTV/nc/nc64
````
# 2. donner les permission executable
````bash 
chmod +x nc64
````
# 3. deplacer dans le dossier binaire 
````bash 
mv nc64 /usr/local/bin/nc
````
# 4. verifier que sa a bien marcher 
````bash 
nc  -h
````
